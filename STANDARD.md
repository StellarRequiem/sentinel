# The Standard

This repository is held to the same measurable, independently-verifiable bar as the
rest of the work — the canonical Standard:
**[StellarRequiem/Forest-Soul-Forge/STANDARD.md](https://github.com/StellarRequiem/Forest-Soul-Forge/blob/main/STANDARD.md)**.

A unit of work ships only at full marks: **Tested** (CI green) · **Verified** (a
VERIFIED block with live proof) · **Audited** (append-only, tamper-evident) ·
**Calibrated** (claimed edges logged + scored vs. base rate; N/A when there is no
predictive claim) · **Honest gaps** (stated, not hidden). Every criterion must
resolve to evidence a third party can re-run or read — CI, `git`, the verification
gate, the public calibration-log. An unverifiable claim does not count.

Maintained pseudonymously (**StellarRequiem**); real-world identity firewalled.
